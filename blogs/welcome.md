This site is intentionally simple and text-heavy.

If you are reading this post, the markdown pipeline is working.

## How to add posts

1. Create a new file in `blogs/` with a `.md` extension.
2. Add frontmatter with `title`, `date`, and optional `tags`.
3. Save the file and refresh the app.

That is it.
